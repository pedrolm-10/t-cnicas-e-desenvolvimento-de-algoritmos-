# 📘 Técnicas e Desenvolvimento de Algoritmos

Este repositório contém materiais, exercícios e projetos desenvolvidos na disciplina **Técnicas e Desenvolvimento de Algoritmos**.  
O objetivo é aplicar conceitos fundamentais de lógica, programação e análise de algoritmos para a solução de problemas computacionais.

---

## 🎯 Objetivos da Disciplina
- Desenvolver o raciocínio lógico aplicado à programação.
- Conhecer e aplicar estruturas de controle de fluxo.
- Implementar algoritmos usando boas práticas de programação.
- Entender conceitos de análise e complexidade de algoritmos.
- Trabalhar com modularização, funções e procedimentos.
- Introduzir estruturas de dados básicas.

---

## 🛠️ Tecnologias e Ferramentas
- **Linguagem:** C / Python / Java (ajustar conforme a disciplina)  
- **Ambiente de Desenvolvimento:** VS Code, Dev-C++, PyCharm, etc.  
- **Controle de Versão:** Git e GitHub
